# Hephaestus

## A template for building discord bots

There are 3 python files included with Hephaestus:
1. Hephaestus.py
2. HephaestusBare.py
3. secret.py

Hephaestus.py and HephaestusBare.py are the exact same program, except Hephaestus.py is full of comments that explain how the code works to the best of my ability :)

For info on how to use the framework, refer to the wiki page [here](https://github.com/MoonriseSunset/Hephaestus/wiki).
