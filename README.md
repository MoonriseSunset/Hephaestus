# Hephaestus

## A template for building discord bots

There are 3 python files included with Hephaestus:
1. Hephaestus.py
2. HephaestusBare.py
3. secret.py

Hephaestus.py and HephaestusBare.py are the exact same program, except Hephaestus.py is full of comments that explain how the code works to the best of my ability :)

secret.py stores the Discord bot's ID, and thus is kept separate from the rest of the program. **If you do republish/fork this code, make sure to REMOVE THE BOTID from secret.py, otherwise others will be able to control your bot**
